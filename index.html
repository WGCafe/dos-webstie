What is dos network?

DOS Network is a layer 2 chain-agnostic Decentralize Oracle Service network, providing real-time data feed oracle and computation oracle solution to mainstream blockchains.

Use case

  • AUTOMATIC INSURANCE PAYOUT

    With DOS Network, many scenarios like flight delay insurance are able to achieve automatic payout by bringing external data and events to the insurance contract.

  • STABLECOINS & CRYPTO DERIVATIVES

    Stablecoins and crypto derivatives require a frequent use of DOS Network to fetch realtime price feed reliably and verifiably.

  • CRYPTO LENDING PLATFORM

    Stablecoins and crypto derivatives require a frequent use of DOS Network to fetch realtime price feed reliably and verifiably.

  • CROSS-CHAIN DEX

    Deploying two DEX contracts on different blockchains and DOS Network acts as an bridge to connect heterogeneous transactions crossing chains.

  • CASINOS AND GAMES

    Secure randomness is the core to blockchain based casinos and games. DOS Network provides unpredictable, verifiable, and secure randomness to drive those applications.

  • COMPUTATION MARKETPLACE

    Commercial computation intensive tasks, such as machine learning model training, 3D rendering, benefit a lot from verifiable off-chain computation market.
