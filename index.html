What is dos network?

DOS Network is a layer 2 chain-agnostic Decentralize Oracle Service network, providing real-time data feed oracle and computation oracle solution to mainstream blockchains.

DOS network features

  • DECENTRALIZED & RELIABLE

    A trustless solution without single point of failure.

  • VRF AND ADVANCED CRYPTOGRAPH

    Driven by Verifiable Random Function and advanced cryptographic signatures

  • REAL-TIME AND VERIFIABLE

    Near real-time request fulfillment and on-chain verifiable response.

  • NON-INTERACTIVE VERIFIABLE COMPUTATION

    Unbounded computation power for smart contracts secured by SNARK-based computation oracle

  • COMPATIBLE TO HETEROGENEOUS BLOCKCHAINS

    Support multiple chains and enable cross-chain interactions.

  • CRYPTO INCENTIVIZED

    Dynamic network incentivized by honest participation reward
